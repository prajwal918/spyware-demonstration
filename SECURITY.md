# Security Policy

## Supported Versions
We currently support the latest major version for security updates.

| Version | Supported          |
| ------- | ------------------ |
| v1.0.x  | :white_check_mark: |
| < v1.0  | :x:                |

## Reporting a Vulnerability
We take security very seriously. If you discover a vulnerability, please do **not** open a public issue. 

Instead, please email the maintainer directly. We will provide a triage response within 48 hours. 

Thank you for helping keep this project secure!
